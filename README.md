# FreelancerOS

Protótipo inicial em React + TypeScript + Tailwind CSS + Recharts.

## Rodar localmente

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
pnpm preview
```

## Stack

- React
- TypeScript
- Vite
- Tailwind CSS v4
- React Router
- Recharts
- Lucide React

## Observação

Este projeto ainda é um MVP visual com dados mockados. Ele já serve como base boa de dashboard, mas ainda não tem:

- backend
- autenticação real
- persistência
- formulários conectados
